# Customer Journey Funnel

## Funnel Stages

1. **Ad Click**
   - GCLID captured at landing.
2. **Website Engagement**
   - Dynamic number insertion swaps phone numbers.
3. **Call Initiation**
   - User calls tracking number.
4. **Call Qualification**
   - Call is tagged and qualified in tracking platform.
5. **Offline Conversion**
   - Conversion uploaded to Google Ads with optional revenue.
