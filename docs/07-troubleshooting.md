# Troubleshooting

## Common Issues

### GCLID Missing
- Verify Google Ads auto-tagging is enabled.
- Confirm landing page does not strip query parameters.

### DNI Not Swapping
- Check that the swap script is loaded on every page.
- Validate that phone number markup matches the platform requirements.

### Conversion Upload Errors
- Confirm conversion action IDs and customer ID.
- Ensure conversion times are within the allowed window.
