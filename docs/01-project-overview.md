# Project Overview

This repository provides a complete reference implementation for Google Ads offline conversion tracking focused on call attribution. It combines GCLID capture, Dynamic Number Insertion (DNI), call tracking platform integration, and conversion uploads to Google Ads.

## Goals

- Connect ad clicks to phone calls and offline revenue.
- Provide ready-to-use templates for scripts and integrations.
- Document the full implementation and testing process.

## Deliverables

- Website DNI templates.
- Google Ads API upload scripts.
- Call tracking platform configurations.
- Testing checklists and sample data.
